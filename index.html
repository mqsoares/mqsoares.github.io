SOBRE

Desenvolvedor Web / Front End, graduando no Curso Superior de Tecnologia em Desenvolvimento Web pela UNOPAR - Universidade Norte do Paraná.

Após dez anos trabalhando com administração de empresas no ramo óptico e alguns anos de barbearia, decidi fazer transição de carreira para a área de TI. Toda experiência acumulada ao longo desses anos contribuiu para desenvolver e fortalecer minhas soft skills, além de propocionar um melhor aproveitamento na evolução das hard skills.

Meu trabalho está diretamente ligado à experiência do usuário, desenvolvo interfaces de sites, assegurando que os elementos gráficos do site fiquem semelhantes aos que foram projetados pelo designer, cuidando de todos os elementos visuais contidos na página e deixando-os totalmente responsivos, ou seja, idependente do tamanho da tela do usuário, o site vai se redimensionar para que assim o usuário consiga ter uma boa experiência de uso.

SKILLS

Tecnologias

HTML

HTML (HyperText Markup Language) "Linguagem de Marcação de Hipertexto". Consiste em uma linguagem de marcação utilizada para produção de páginas na web.

CSS

CSS (Cascading Style Sheets) são folhas de estilo em cascata. linguagem que complementa e formata o HTML, Com ele, podemos modificar praticamente tudo dentro do layout (Cores, background, margens, preenchimentos e etc.).

JavaScript

É uma das mais importantes tecnologias voltadas para o front-end, unindo-se ao HTML e CSS, abrangem praticamente todas as exigências do desenvolvimento de uma página completa, dinâmica e com boa performance.

React

É uma biblioteca front-end e tem como um de seus objetivos facilitar a conexão entre diferentes partes de uma página, portanto seu funcionamento acontece através do que chamamos de componente.
OBS. Iniciando os estudos nessa "ferramenta".

Git

É um Sistema de Controle de versão que evita que alterações realizadas em um projeto modifiquem o código-fonte. Assim, caso haja algum problema em uma alteração do código-fonte, é possível restaurar a versão anterior de forma fácil e rápida.
